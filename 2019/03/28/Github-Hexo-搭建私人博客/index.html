Github + Hexo 搭建私人博客

发表于 2019-03-28

要素总览

  • Windows 10 64bit
  • Node.js
  • git & Github
  • hexo

    步骤

  1. 安装 Node.js（https://nodejs.org/en/） 并配置好 Node.js 的环境。如果配置成功，那么我们在命令行中输入 node -v 就会出现如下界面

  2. 同样的，安装 git（https://git-scm.com/downloads） 也同样如此，此处不再赘述。检查 git 是否安装成功只需要在命令行中输出 git -v 就可以了。

  3. Github账户注册和新建项目，项目必须要遵守格式：[账户名].github.io，不然接下来会有很多麻烦。并且需要勾选Initialize this repository with a README

  4. 仓库创建完毕之后转到仓库的 Setting 之下，拉到后面就会发现我们的仓库对着一个网址。

  5. 安装 Hexo ，以下是命令：

由于 Hexo 服务开启的默认端口是 4000 ，所以成功之后登上这个主机端口，浏览器就会显示如下图片

  1. 由于我们利用 Github 进行站点部署，所以本机开发完了之后需要将文件发送至 Github 的服务器上。但是有一点就是机器之间的受信问题。此处我们采用 SSH 认证的方式建立 Github 服务器与我们自己机器信任关系。这样一来我们就无需在上传文件的时候输入密码了。

  2. 登录 Github 之后，点击头像下的 Setting 添加 SSH。如果之后我们利用 Git Bash 成功登录到 GitHub，就代表我们的认证方式配置成功。输入命令 ssh -T git@github.com。
    如果看到Hi后面是你的用户名，就说明成功了。

  3. 配置工程根目录下的 _config.yml ，修改 repo 的值。此处的 repoistory 是仓库的 SSH 地址

  4. 新建博客 hexo new post “博客名”

  5. 编写完此页之后，输入命令 npm install hexo-deployer-git --save 安装扩展

  6. 输入命令 hexo d -g 部署
